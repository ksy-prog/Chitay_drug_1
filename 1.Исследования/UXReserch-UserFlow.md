# User Flow — Пользователь

| Экран | Действие | Основное содержание экрана | Навигация |
|---------|---------|---------|---------|
| Онбординг | Начало работы | Приветствие | Кнопка «Пропустить» или последний экран онбординга |
| Экран входа | Войти | Минимальная форма входа | Кнопка «Войти» |
| Экран регистрации | Ввести личную информацию | Ввод данных, кнопка «Зарегистрироваться» | Из онбординга |
| Экран восстановления пароля | Забыл пароль | Введите ваш e-mail адрес, на который зарегистрирован аккаунт, и мы вышлем код для подтверждения личности | Под формой «Авторизации» |
| Экран ввода кода | Ввёл e-mail | Код otp с возможностью отправить код еще раз | Кнопка «Получить код» |
| Экран придумайте новый пароль | Ввёл код | New пароль + выйти со всех устройств Да/Нет (чекбокс) | Кнопка «Войти» |
| Главный экран | Начинает чтение / Просмотр метрик | Личный кабинет в виде дашборда с основными метриками пользователя | Кнопка «Начать чтение», кнопка «Продолжить...» |
| Выбор текста | Выбирает нужный текст | Карточки текстов, поисковая строка, фильтрация, сортировка | Клик по тексту, кнопка «Назад» |
| Задание | Знакомится с инструкцией | Задание перед чтением, короткое объяснение, что делать, в дружеской форме с персонажами | Назад к текстам, кнопка «Начать» |
| Чтение | Читает | Фрагментация текста, прогресс при чтении | Кнопка «Я прочитал», кнопка «Продолжить читать», кнопка «Назад» |
| Диалог с ИИ | Отвечает на вопросы по пройденному материалу | Вопросы + поле ответа, подсказки, анимация/видео ИИ-ассистента, в дружеской форме с персонажами в начале и конце текста | Кнопка «Ответить», следующий вопрос (внутри экрана) |
| Переход к следующему фрагменту | Завершение блока и хочет продолжить читать следующий фрагмент | Переход к следующему фрагменту | Кнопка «Продолжить читать» |
| Завершение текста | Заканчивает текст | Сообщение о завершении | Кнопка «Далее» |
| Рефлексия | Формулирует вывод | Открытые вопросы, анимация/видео ИИ-ассистента | Кнопка «Завершить» |
| История чтения | Смотрят прошлые сессии | Новые метрики после прочтения, список прочтенной литературы, основные метрики | Кнопка «Посмотреть историю» / «Назад» |
| Ошибка | Видит сообщение об ошибке | Нет данных / ошибка загрузки | Кнопка «Повторить», возврат на предыдущий экран, кнопка «На главную» |

## Примечания

- Завершить работу можно с любого экрана, просто закрыв приложение. Тогда же завершится и сеанс пользования.
- Выйти на главный экран приложения можно с любого экрана приложения.
- Попасть на страницу 404 можно с любого экрана (если ошибка).
- В User flow добавить систему подсказок.
- Что надо делать.
- Сделать стимуляцию / мотивацию пользователя читать.

# User Flow — Администратор

| Раздел | Действие | Содержание | Навигация / Функции |
|---------|---------|---------|---------|
| Админ-панель (дашборд) | Управляет системой | Статистика платформы, Пользователи, Тексты, Метрики | Центральная точка управления |
| Управление контентом | Просматривает тексты и упражнения | Список материалов, фильтры, сортировка | Добавить, Редактировать, Удалить |
| Добавить материал | Добавить | Загрузка материала | Кнопка «Добавить (иконка)» |
| Редактирование материала | Редактировать | Редактирование материала | Кнопка «Редактировать (иконка)» |
| Удаление материала | Удалить | Подтверждение (Да/Нет) | Кнопка «Удалить (иконка)» |
| Сохранение изменений | Сохранить | Подтверждение изменений | Кнопка «Сохранить» |
| Пользователи | Выбрать пользователя | Список пользователей | Переход в профиль пользователя |
| Профиль пользователя | Просматривает данные пользователя | Прогресс, активность, история | Изменить роль, Заблокировать, Назад |
| Метрики | Анализирует показатели | Активность, удержание, ошибки | Просмотр аналитики |
| Настройки | Настраивает систему | Уровень сложности, логика вопросов, параметры ИИ | Сохранить изменения |
| Pop-up окно сохранить | Подтверждает изменения | Да | Сохранение |
| Добавить Админа | Создание администратора | Новый администратор | Добавление |

## Роль администратора

**Функция:** управление контентом, пользователями и системой.
