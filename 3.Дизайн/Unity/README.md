# ЧитайДруг — Интерактивный голосовой помощник для чтения

Десктопное приложение на Unity, которое слушает чтение пользователя вслух, распознаёт речь, генерирует короткие реплики персонажа через языковую модель и озвучивает их. Создано для поддержки детского чтения в игровой форме.

## 🧠 Как это работает

1. Пользователь читает текст вслух.
2. Система записывает аудио (микрофон).
3. Определяет паузы и окончание чтения (SilenceDetector).
4. Преобразует аудио в текст (Vosk offline).
5. Отправляет текст в локальную языковую модель (Ollama).
6. Генерирует поощряющий или поддерживающий ответ персонажа.
7. Озвучивает ответ через синтез речи Windows (SpeechLib / System.Speech).
8. При длительном молчании инициирует мягкие подсказки.

## 🔄 Состояния системы (State Machine)

Схема состояний:
Listening → Recording → Processing → Speaking → Listening
↑
PausedHelp

### Описания состояний:

- **Listening** — ожидание речи, микрофон включён.
- **Recording** — активная речь, запись аудио.
- **Processing** — остановка записи, распознавание (Vosk) и генерация ответа (Ollama).
- **Speaking** — озвучивание ответа.
- **PausedHelp** — долгая пауза (5 сек), подсказка персонажа.

> Все сервисы общаются только через центральный `SessionManager`.

| Язык                      | C#                                                               |
| Распознавание речи (STT)  | [Vosk](https://alphacephei.com/vosk/) (офлайн, русский язык)     |
| Языковая модель (LLM)     | [Ollama](https://ollama.com/) (локальный сервер, OpenAI‑совместимый API) |
| Синтез речи (TTS)         | System.Speech.dll / SpeechLib (Windows, русские голоса)          |
| Версионирование           | Git + GitHub                                                     |
